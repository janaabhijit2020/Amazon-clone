🛒 Amazon Clone

A responsive front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. This project replicates the layout and design of Amazon's landing page to practice web development fundamentals such as page structure, styling, Flexbox, and responsive design.

---

## 📖 Overview

This project is a static clone of the Amazon homepage created for learning and practicing front-end development. It focuses on recreating the user interface and improving HTML and CSS skills.

> **Note:** This project is for educational purposes only and is not affiliated with or endorsed by Amazon.

---

## ✨ Features

- Responsive navigation bar
- Amazon-style search bar
- Hero banner section
- Product category cards
- Footer with multiple sections
- Hover effects on buttons and links
- Clean and organized layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
